# Tornado X

Add engineering suites for your projects.

## Usage

With npm:

```bash
$ npm i -g tornado-x
$ tornado-x -v
```

With yarn:

```bash
$ yarn -g tornado-x
$ tornado-x -v
```

With pnpm:

```bash
$ pnpm i -g tornado-x
$ tornado-x -v
```
