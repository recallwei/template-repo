# Create Tornado

## Install

```bash
# With npm
$ npm create tornado@latest
# With yarn
$ yarn create tornado
# With pnpm
$ pnpm create tornado
```

## With a specific template

```bash
# With npm
$ npm create tornado@latest --template [template-name]
# With yarn
$ yarn create tornado --template [template-name]
# With pnpm
$ pnpm create tornado --template [template-name]
```
